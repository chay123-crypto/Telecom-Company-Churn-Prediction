# Telecom-Company-Churn-Prediction
This project focuses on predicting customer churn in a telecom company using exploratory data analysis and machine learning techniques.
It analyzes customer behavior, service usage, and billing patterns to identify key factors influencing churn. 
The classification model helps businesses take proactive steps to improve customer retention.
Conducted Thorough Exploratory Analysis and used Simple Logistic Regression Model with 80% score with accuracy metrics
